Welcome to Diatom

Most browsers let you see the internet. Diatom is a filter — it keeps the internet from seeing you. Your browsing history, 2FA codes, and archived pages never leave your device.

This wizard takes about 2 minutes. You can change everything later in Settings.

Local AI (optional)

Diatom can run AI on your device — no cloud, no API key. Queries never leave your machine. This requires Ollama to be installed.

Checking for Ollama…

Install Ollama from ollama.ai, then click Refresh. You can skip this step and add it later.

Choose your privacy posture

All modes protect your privacy. This sets how aggressively Diatom reconfigures the web for you. You can tune individual features anytime.

⚖ Balanced (recommended)
Blocks trackers, injects fingerprint noise, keeps most sites working
🔒 Strict
All balanced features + timezone spoofing + Zen mode default on
🪶 Minimal
HTTPS upgrade and tracker block only — maximum compatibility

Filter rules (optional)

Diatom's built-in blocklist catches analytics and trackers. Subscribe to community-maintained lists for broader coverage. Diatom downloads, you choose.
